TAVNO - Fullstack ZIP
======================

Fichiers inclus:
- package.json
- server.js
- db.json
- /public/index.html
- /public/style.css
- /public/app.js
- /uploads/sample-1.jpg
- /uploads/sample-2.jpg

Installation (local):
1. Node.js v16+ recommandé.
2. `npm install`
3. `npm start`
4. Ouvrir http://localhost:3000

Notes:
- mot de passe du user seed : jean@example.com / password
- lowdb (db.json) sera initialisé automatiquement au démarrage
- pour production, utiliser une vraie base de données et sécuriser l'authentification
